## Steps in Windows 

On command prompt:

Create a directory 
```
$ mkdir appOmniStack/backend
```

Enter in this dir. and init a yarn (a new file packege.json will be criete)
```
$ cd appOmniStack/backend
$ yarn init -y
```

Open the node.js command promtp, enter in the **same directory** and write: 
```
$ yarn add express
```
